# Kotlin-AdministradorAlimentosPrueba
Aplicación de prueba en la cuel se permite realizar registro de personas, así como logeo, y guardado de cada una de sus comidas diarias. La aplicación guarda dichos datos en SQLite o Firebase, dependiendo de la disponibildad de internet. También se utiliza una API para mostrar una bebida aleatoria una vez ingresada cada comida. En esta prueba se utilizó Material Design, SQLite, ViewPager2, Firebase, RoboElectric para la realización de los tests de la DB, Junit y Espresso para testeos automatizados, Retrofit y Gson para el llamado de la API y Glide para el uso de imágenes.
